# webpack

## concepts
- 모던 자바스크립트 어플리케이션을 위한 static module bundler(정적 모듈 번들러)
- webpack이 어플리케이션에서 처리되면 내부적으로 프로젝트에서 필요로 하는 모든 모듈들을 dependency graph로 빌드하고 하나 이상의 번들을 생성한다.
